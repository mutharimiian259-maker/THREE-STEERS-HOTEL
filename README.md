# 🏨 Three Steers Hotel Meru Website

A modern, SEO-optimized hotel website for **Three Steers Hotel, Meru Kenya**, built with **Next.js App Router + TailwindCSS**.

Designed to maximize:
- Direct WhatsApp bookings 📲  
- Google Search visibility 🔍  
- Mobile-first performance 📱  

---

## 🚀 Live Features

- ⚡ Next.js App Router architecture  
- 📱 Fully responsive mobile-first UI  
- 🔍 SEO optimized (Google Hotel searches)  
- 💬 WhatsApp booking integration  
- 📞 Click-to-call functionality  
- 🏨 Rooms, dining, conference, and experience showcase  
- 🗺 Google Maps integration  
- 📰 Blog system for SEO traffic growth  

---

## 🧠 Tech Stack

- Next.js (App Router)
- React
- TailwindCSS
- Google Maps Embed API
- WhatsApp Click-to-Chat API

---

## 📂 Project Structure

/app → Pages & routes (App Router)

/data → Static content (rooms, blogs)
/public → Images & assets
